# Claude Code Discord Presence

A VSCode extension that shows **Discord Rich Presence** whenever you're using Claude Code.

---

## Features

- Detects Claude Code via terminal and OS process scanning
- Updates Discord profile in real-time with session duration and Claude Code icon
- Status bar indicator shows active state and Discord connection
- Auto-reconnects if Discord restarts

---

## Requirements

- Discord must be running on your machine
- A Discord Application with `claude_code` image uploaded to Rich Presence Art Assets

## Development

```bash
git clone https://github.com/ashutoshswamy/claude-code-discord-presence
cd claude-code-discord-presence
npm install
npm run compile

# Press F5 in VSCode to launch Extension Development Host
```
